# PGPS UDP Packet Listener

Listens for poses coming from PGPS and writes them to a CSV file.

**WARNING: This is very rudimentary and unpolished**

## Compile and Run

- Compile `nob`: `gcc nob.c -o nob` (only have to do this once)
- Compile and run `pgps_listener`: `./nob && ./build/pgps_listener`
